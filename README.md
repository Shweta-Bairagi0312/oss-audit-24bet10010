## Project Description
This project is a capstone submission for the **Open Source Software (OSS NGMC)** course.  
The chosen software for this audit is **Python**, an open-source programming language licensed under the **Python Software Foundation (PSF) License**.  

The project contains:  
- A detailed report on Python's origin, license, Linux footprint, ecosystem, and comparison with proprietary software.  
- Five shell scripts demonstrating Linux and Bash scripting concepts.  

---

## Files Included

### 1. `script1.sh` — System Identity Report
**Displays:**  
- Linux distribution  
- Kernel version  
- Logged-in user  
- Home directory  
- Uptime  
- Date and time  
- Open source license information  

### 2. `script2.sh` — FOSS Package Inspector
**Checks:**  
- Whether Python is installed  
- Installed version  
- Binary location  
- License-related details  
- Includes a case statement for package philosophy notes  

### 3. `script3.sh` — Disk and Permission Auditor
**Audits:**  
- Important Linux system directories  
- Permissions  
- Ownership  
- Disk usage  
- Python-specific installation directories  

### 4. `script4.sh` — Log File Analyzer
**Reads a log file and:**  
- Searches for a keyword  
- Counts matches  
- Displays matching lines  
- Shows summary statistics  

### 5. `script5.sh` — Open Source Manifesto Generator
**Functionality:**  
- Asks the user three interactive questions and generates:  
  - A personalised open source manifesto  
  - Saves output to a `.txt` file
